# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-05

### Added

- Page framework with registration DSL, routing, and rendering
- Page versioning with deploy-oriented version tracking
- SEO metadata (meta_title, meta_description, Open Graph tags)
- A/B testing with experiments, variants, and weight-based assignment
- Experiment lifecycle management (draft, running, paused, completed)
- Funnel tracking with step-by-step visitor progression
- Funnel results with visitor counts and drop-off rates
- Lead capture with automatic A/B test conversion tracking
- Pluggable event system (Log, Null, and Ahoy adapters)
- Admin UI for pages, experiments, funnels, and leads
- Admin authentication hook via configuration
- Install generator with initializer template and migrations
- Configuration DSL for site settings, auth, and event adapters
